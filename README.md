# Jekyll Site Template
Repository template for jekyll static sites.

If you're about to start a new site, create a new repository from this template using the button above:

![](https://user-images.githubusercontent.com/1926652/82120678-dba6bd80-977f-11ea-8390-bae6dff20b91.png)

If you already have an existing repository then simply copy the `.github/workflows/deploy.yml` file into your repository and push to GitHub.
